## **5. PRIVACY POLICY SUMMARY**

### **What's Collected:**

**Both Apps:**

```
PERSONAL INFORMATION
- Name, phone number, email
- Profile photo (optional)
- Government ID (drivers only - for verification)
- Vehicle documents (drivers only)

LOCATION DATA
- Real-time GPS location (only during active trips)
- Pickup and destination addresses
- Trip route history

USAGE DATA
- Trip history and frequency
- App interactions and preferences
- Device information (model, OS version)
- Crash reports and performance data

PAYMENT DATA
- Subscription payment records
- Transaction history
- Payment method information (processed securely)

COMMUNICATIONS
- In-app messages with drivers/riders
- Support conversations
- Push notification preferences

```

### **How Data is Used:**

```
- Match riders with nearby drivers
- Provide navigation and routing
- Process subscription payments
- Safety and security monitoring
- Customer support
- Platform improvements
- Legal compliance

```

### **Data Sharing:**

```
- Driver/rider details shared for trip coordination
- Emergency contact notifications (when SOS activated)
- Payment processors (for subscriptions)
- Legal authorities (when required by law)
- Never sold to third parties

```

### **Data Retention:**

```
- Active account data retained while account is active
- Trip history kept for 2 years
- Location data deleted after 90 days (except safety records)
- Users can request data deletion at any time

```
